Projeto Banco de Dados

Projeto Banco de Dados

×

Este projeto foi realizado como projeto final da matéria de Programação Orientada a Objetos, na graduação. Tive como instrução a criação de 3 tabelas em um banco de dados utilizando a plataforma PostgreSQL, onde fiz uma aplicação simulando uma loja de componentes eletrônicos. Na criação das tabelas, desenvolvi uma tabela para registro de clientes, uma tabela para registro de componentes e uma tabela para registro das vendas realizadas "pela loja". Após a criação do banco, fiz um programa pelo Microsoft Visual Studio, na ferramenta Aplicativo do Windows Forms, onde fiz uma aplicação gráfica e fiz um sistema de CRUD para registro de cada parte do meu banco de dados.
